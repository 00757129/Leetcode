# 題目大意：
計算 unsigned int 中（2 進位表示） 1 出現的個數。

# 解題過程：
  1. 以迴圈從 n 的尾端開始檢查每個 bit
    （使用 while，而非 for 時，可以順便在 n 已無 1 時，結束迴圈）
  2. 若 n & 1 ==1，則表示 n 的最後一個位數為 1，count++
--------------------
時間複雜度：O(N)，N 為 n 的 bit 數
空間複雜度：O(1)
